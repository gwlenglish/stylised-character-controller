* Setup
---------------------------

if you want to use your controller for camera rotating & zooming.
Here is my configuration using XBox 360 controller on Windows:
	Edit > Project Settings > Input > Axes
		RScrollX
			Name		RScrollX
			Gravity 	0
			Dead		0.19
			Sensitivity	1
			Type		Joystick Axis
			Axis		4th axis(Joysticks)
			JoyNum		Get Motion from all Joysticks
		RScrollY
			Name		RScrollY
			Gravity 	0
			Dead		0.19
			Sensitivity	1
			Type		Joystick Axis
			Axis		5th axis(Joysticks)
			JoyNum		Get Motion from all Joysticks
		Trigger
			Name		RScrollX
			Gravity 	0
			Dead		0.019
			Sensitivity	0.1
			Type		Joystick Axis
			Axis		3rd axis(Joysticks)
			JoyNum		Get Motion from all Joysticks
